By default, the ndi.calc.example.simple calculator searches for all NDI documents in the session.

Default Search Criteria:
- Searches for any document that is a subclass of 'base' (which includes all NDI documents).
- The dependency name is set to 'document_id'.

Modifying the Search:
To restrict the operation to a specific subset of documents, modify the 'query' field of the search parameters.

Example:
% Only perform the operation on documents with the name 'my_experiment_data'
DQ = c.default_search_for_input_parameters();
DQ.query = DQ.query & ndi.query('base.name', 'exact_string', 'my_experiment_data', '');
c.run('Replace', DQ);
