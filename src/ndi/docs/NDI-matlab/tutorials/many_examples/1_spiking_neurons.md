# Tutorial 5.1: Spiking neurons

## Finding Spiking Neurons

If you have an `ndi.session` object called `session`, you can find all the spiking neurons by calling:

```matlab
spikingNeurons = session.getelements('element.type','spikes');
```

## Finding Spiking Neurons Recorded by a Probe

You can also find spiking neurons recorded by a specific probe (`myProbe`) using the database search and converting the document back to an object:

```matlab
q1 = ndi.query('element.type','exact_string','spikes');
q2 = ndi.query('','depends_on','underlying_element',myProbe.id());
spikingNeuronDocs = session.database_search(q1&q2);
spikingNeurons = {};
for i=1:numel(spikingNeuronDocs)
    spikingNeurons{i} = ndi.database.fun.ndi_document2ndi_object(spikingNeuronDocs{i},session);
end
```

## Finding and Plotting Neuron Extracellular Documents

You can find the `neuron_extracellular` documents that are associated with a specific spiking element (such as one of the `spikingNeurons` found above) by querying the database for documents that depend on the `element_id`.

Here is a comprehensive example function that demonstrates how to find these documents and plot the mean waveforms.

```matlab
function neur_extracell(session, mySpikingNeuron)
% For accessing neuron_extracellular data from NDI Database
% for plotting mean waveforms generated by JRCLUST
%
%  NEUR_EXTRACELL(SESSION, MYSPIKINGNEURON)

    % 1. Search for 'neuron_extracellular' documents for a specific element
    q1 = ndi.query('', 'isa', 'neuron_extracellular');
    q2 = ndi.query('', 'depends_on', 'element_id', mySpikingNeuron.id());
    docs = session.database_search(q1 & q2);

    if isempty(docs)
        error('No neuron_extracellular documents found for the given element.');
    end

    % 2. Setup Plotting Parameters (4x4 Grid)
    numUnits = numel(docs);
    unitsPerPage = 16;  % 4x4 layout
    numFigs = ceil(numUnits / unitsPerPage);

    for f = 1:numFigs
        figure('Name', sprintf('Mean Waveforms Group %d', f), 'Color', 'w', 'Position', [100 100 1400 900]);

        startIdx = (f-1) * unitsPerPage + 1;
        endIdx = min(f * unitsPerPage, numUnits);

        for i = startIdx:endIdx
            subplotIdx = mod(i-1, unitsPerPage) + 1;
            subplot(4, 4, subplotIdx); % 4x4 Grid

            % Access the properties of the document directly
            props = docs{i}.document_properties.neuron_extracellular;

            % Extract Data
            wf = props.mean_waveform;
            t_data = props.waveform_sample_times;
            quality = props.quality_label;
            clustIdx = props.cluster_index;

            % Handle time conversion to ms
            % Assuming t_data is in seconds
            if numel(t_data) == 1
                t_plot = (0:length(wf)-1) * t_data * 1000;
            else
                t_plot = t_data * 1000;
            end

            % Determine Plot Color based on Quality
            plotColor = [0.4 0.4 0.4]; % Default Gray
            if strcmpi(quality, 'Single') || strcmpi(quality, 'SUU')
                plotColor = [0 0.4470 0.7410]; % Blue for Single Unit
            elseif strcmpi(quality, 'Multi') || strcmpi(quality, 'MUA')
                plotColor = [0.8500 0.3250 0.0980]; % Orange/Red for Multi
            end

            % Plot against time in ms
            plot(t_plot, wf, 'LineWidth', 1.5, 'Color', plotColor);

            % Formatting
            title(sprintf('Cluster: %d\nType: %s', clustIdx, string(quality)), ...
                'FontSize', 8, 'Interpreter', 'none');
            xlabel('Time (ms)');
            ylabel('Voltage');
            grid on;
            axis tight;
        end

        sgtitle(sprintf('Session: %s | Units %d to %d', session.reference, startIdx, endIdx));
    end
end
```
